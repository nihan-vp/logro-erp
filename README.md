## Installation
1. Clone the repository: `git clone https://github.com/nihan-vp/logro-erp.git`
2. Navigate to the project directory: `cd logro-erp`
3. Install dependencies: `npm install`

## Running the Project
1. Start the development server: `npm run dev`
